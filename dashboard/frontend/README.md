# AI Interviewer Dashboard

This project is a React-based dashboard for visualizing AI In-depth Interview insights, specifically "Coding Table" data.

## Prerequisites

- [Node.js](https://nodejs.org/) (Version 16 or higher recommended)

## Setup & Running

1.  **Navigate to the project directory**:
    ```bash
    cd frontend
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```
    *Note: This downloads the necessary libraries (React, Recharts, Lucide, etc).*

3.  **Start the Development Server**:
    ```bash
    npm run dev
    ```

4.  **View the Dashboard**:
    - Open your browser and go to the URL shown in the terminal (usually `http://localhost:5173`).

## Build for Production

To create a production-ready build:

```bash
npm run build
```

The output will be in the `dist` folder.
